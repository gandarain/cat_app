# Cat App
This is an application to show list of cat from this API

https://documenter.getpostman.com/view/5578104/RWgqUxxh

### How to run
- `yarn install`
- `cd ios && pod install`
- create a new file for `.env`
- copy `.env.example` to `.env`
- `yarn start`
- `yarn ios`
